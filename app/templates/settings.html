🧠 How to get an Anthropic API Key

The Anthropic API is pay-as-you-go. Ascent uses the Haiku model which costs around $0.001 per coaching conversation — a $5 credit lasts months of typical use.

Step 1 — Create an Anthropic account

  1. Go to console.anthropic.com
  2. Click Sign up and register with your email or Google account
  3. Verify your email address

Step 2 — Add a payment method and buy credits

  1. Once logged in, click your profile in the top right → Billing
  2. Click Add payment method and enter a credit card
  3. Click Buy credits — $5 is plenty to start
Credits must be purchased before the API will work. The free tier does not include API access.

Step 3 — Generate an API key

  1. In the console, go to Settings → API Keys
  2. Click Create Key
  3. Give it a name (e.g. Ascent)
  4. Copy the key immediately — it starts with sk-ant- and will only be shown once

Step 4 — Paste it here

Paste the key into the field above and click Save Key. You'll need to restart Ascent once for it to take effect.

Your key is stored locally in ~/.ascent_config on your Mac. It is never sent anywhere except directly to Anthropic's API.
