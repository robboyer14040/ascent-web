🧠 How to get an Anthropic API Key

The Anthropic API is pay-as-you-go. Ascent uses the Haiku model which costs around $0.001 per coaching conversation — a $5 credit lasts months of typical use.

Step 1 — Create an Anthropic account

  1. Go to console.anthropic.com
  2. Click Sign up and register with your email or Google account
  3. Verify your email address

Step 2 — Add a payment method and buy credits

  1. Once logged in, click your profile in the top right → Billing
  2. Click Add payment method and enter a credit card
  3. Click Buy credits — $5 is plenty to start
Credits must be purchased before the API will work. The free tier does not include API access.

Step 3 — Generate an API key

  1. In the console, go to Settings → API Keys
  2. Click Create Key
  3. Give it a name (e.g. Ascent)
  4. Copy the key immediately — it starts with sk-ant- and will only be shown once

Step 4 — Paste it here

Paste the key into the field above and click Save Key. You'll need to restart Ascent once for it to take effect.

Your key is stored locally in ~/.ascent_config on your Mac. It is never sent anywhere except directly to Anthropic's API.

🟠 How to get Strava API Keys

Strava API access is free. You just need to create a Strava API "application" (takes about 5 minutes) to get your Client ID and Secret.

Step 1 — Log in to Strava

Go to strava.com/settings/api and log in with your Strava account.

Step 2 — Create a Strava API Application

  1. Fill in the form with any reasonable values:
    • Application Name: Ascent (or anything you like)
    • Category: Data Importer
    • Club: leave blank
    • Website: https://ascent-web.fly.dev
    • Authorization Callback Domain: ascent-web.fly.dev
  2. Check the box to agree to Strava's API agreement
  3. Click Create

Step 3 — Copy your keys

After creating the app, Strava shows you:

  • Client ID — a short number (e.g. 12345)
  • Client Secret — a long hex string

Copy both and paste them into the fields above, then click Save Strava Keys.

Step 4 — Connect your account

After saving and restarting Ascent, go to Strava Sync in the top navigation to authorise Ascent to access your activities. You'll be redirected to Strava to approve it, then brought back to start syncing.

Each Strava user who wants to use Ascent needs their own API keys — you can't share yours with someone else's Strava account.
